## About

A app to practice in-app purchase. InspoQuotes is a freemium app that gives some free inspiration and a lot of paid inspiration.



### What i do

* How to setup new in-app purchases on App Store Connect
* How to use the StoreKit API to allow users to purchase premium content
* How to use the StoreKit API to detect failure cases
* How to use the StoreKit API to restore in-app purchases
* How to persist the state of users' purchases

## Installation

```git
git clone https://github.com/iury0393/inspo-quotes
open the folder in the terminal and do a pod install
open the file .xcworkspace
build the application
```
